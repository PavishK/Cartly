## âš™ï¸ Installation

### Frontend (`cartly-app`)

```bash
git clone https://github.com/PavishK/Cartly.git
cd cartly-app
npm install
```

Create a `.env` file:

```env
EXPO_PUBLIC_SERVER_URL="http://localhost:8080"
```

Run the app:

```bash
npx expo start
```

---

### Backend (`cartly-server`)

```bash
cd cartly-server
npm install
```

Create a `.env` file:

```env
MONGODB_URI="your-mongodb-uri"
PORT=8080
AUTH_SECRET="your-auth-secret"
GEMINI_API_KEY="your-gemini-api-key"
```

Start the server:

```bash
npm start
```
